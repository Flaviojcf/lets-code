### Pre-requisites


- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/flaviojcf/lets-code.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
